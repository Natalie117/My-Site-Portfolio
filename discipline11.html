Основи мови SQL

Дисципліна «Основи мови SQL» присвячена вивченню основ реляційних баз даних та мови структурованих запитів SQL (Structured Query Language). Студенти отримують теоретичні знання та практичні навички роботи з базами даних, включаючи створення, модифікацію та видалення таблиць, маніпулювання даними, виконання запитів для отримання потрібної інформації та забезпечення цілісності даних.
